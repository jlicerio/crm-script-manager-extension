# Icons Directory

This directory should contain the extension icons. You'll need to add:

- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels) 
- `icon128.png` (128x128 pixels)

You can create these using any image editor or online icon generator. The icons should represent a script/code manager theme.

## Quick Icon Creation

1. **Use an online favicon generator** like favicon.io
2. **Use the emoji ⚙️ or 🔧** as base
3. **Or create simple geometric icons** with tools like Canva

## Temporary Solution

For now, the extension will work without icons, but Chrome will show a default gray icon.
