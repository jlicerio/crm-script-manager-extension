💬 CRM Chat Assistant

Standalone test version for development and preview

📋 Test Features:

  • Drag & Drop: Click and drag the header to move the chat bubble
  • Auto-responses: Try messages like "hello", "help", "search", "date"
  • Typing indicator: Shows while processing responses
  • Message history: Scroll through conversation
  • Responsive design: Works on different screen sizes

🔧 Integration Notes:

This chat assistant is integrated into the CRM Tools Extension and will appear as a floating bubble when accessed through the tools widget on CRM pages.

🤖 Poe API Integration:

The assistant uses Poe's Embed API to communicate with the @Helpdesk-Assist-v1 bot:

  • Live API: When embedded in a Poe canvas app, uses real AI responses
  • Fallback: When Poe API unavailable, uses pre-programmed responses
  • Streaming: Supports real-time response streaming from the bot

Note: This standalone version will use fallback responses unless loaded within a Poe environment.
