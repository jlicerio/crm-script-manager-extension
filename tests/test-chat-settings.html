🤖 CRM Chat Assistant

Advanced AI Assistant with Multi-API Support & Settings Configuration

🎯 New Features in This Version:

🤖 Poe OpenAI-Compatible API

Integrated with your custom Helpdesk-Assist-v1 bot using Poe's REST API endpoints.

Endpoint: https://api.poe.com/v1/chat/completions

⚙️ Settings Modal

User-friendly interface to configure API keys for Poe, OpenAI, and Anthropic APIs.

Includes status indicators and persistent storage

🔄 Multi-API Fallback

Automatic fallback system: Poe → OpenAI → Anthropic → Local AI → Keywords

Ensures responses even if primary API is unavailable

💾 Persistent Storage

API keys are securely stored in localStorage and automatically loaded on startup.

No need to re-enter keys after browser restart

🧪 Testing Instructions:

  1. Launch Assistant: Click "Launch Chat Assistant" to open the floating chat bubble
  2. Configure APIs: Click "Open Settings" or the ⚙️ button in chat header to set up API keys
  3. Test Connection: Use "Test API Connection" to verify your API setup
  4. Chat Testing: Try messages like:
    • "Hello, how can you help me?"
    • "What CRM features are available?"
    • "Help me with customer data analysis"
    • "Show me reporting options"
