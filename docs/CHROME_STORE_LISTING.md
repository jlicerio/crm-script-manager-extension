## Detailed Description

Boost your CRM productivity with essential tools and smart search enhancements!

**🔧 KEY FEATURES:**

• **Smart Call Dependency Automation** - Learn what you select, auto-fill dependent fields
• **Floating Tools Widget** - Quick access to daily utilities with copy date/URL
• **Enhanced Model Search** - Add powerful search to dropdown selectors  
• **Visual Feedback** - Toast messages and button feedback for all actions
• **Clean, Modular Design** - Minimal interface, easily extensible

**⚡ PRODUCTIVITY BOOST:**

**Call Category Auto-Fill**: Select a Call Reason once, and the extension learns which Category and Action go with it. Next time you pick that Reason, it auto-fills for you. No configuration needed—it learns from your behavior.

**Smart Dropdown Search**: Transform tedious dropdown scrolling into instant results. Type "DLA20" to find "DLA-20LTD" instantly, or search any part of a model number for quick selection.

**🎯 PERFECT FOR:**

• SuiteCRM support teams handling high call volumes
• CRM users tired of manually selecting dependent fields
• Teams needing quick date/URL copying for documentation
• Anyone who wants smarter form interactions
• Users who appreciate learning automation (no complex setup)

**🔒 PRIVACY FOCUSED:**

• No data collection or tracking
• Works entirely offline
• All functionality is local to your browser
• Zero external server communication

**✨ SMART FEATURES:**

• **Learned Mapping System** - Remembers your field selections and auto-fills next time
• **Flexible Search** - Handles dashes, spaces, and partial matches in any dropdown
• **Toast Notifications** - Clear visual feedback for all actions
• **Zero Configuration** - Works automatically, learns from your behavior
• **Pure localStorage** - All data stays in your browser, never sent anywhere

**INSTALLATION:**
Simply install and visit your SuiteCRM site. The floating tool button appears automatically in the bottom-right corner. No setup required!
